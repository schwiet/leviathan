This server only supports https. However, an official certificate has not been purchased (we are a low-budget shop here).

There is a self-signed SSL certificate for this website available for download:

  • crt format (i.e. MS Windows)
  • cer format (?)
  • pem format (i.e. Safari/iOS)

Open this file and install it into your Trusted Root Certification Authority/Certificates store in order to access the https version of this site with no warnings.

You must import the cert into the Trusted Root Certification Authority. You must explicitly select this store. Do not allow it to automatically select which store to use; it will choose the wrong store.

This is sufficient for Internet Explorer and Chrome; however, Firefox requires an additional step of adding a security exception.

Once you've installed my self-signed root cert, the https version of this site is here.
